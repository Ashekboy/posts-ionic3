#### Procedimentos para Instalação

````
$ git clone https://github.com/wellsm/posts-ionic3.git
````

Na raíz do projeto execute:

````
npm install
````

#### Configuração do FIREBASE

````
export const FIREBASE_CREDENTIALS = {
    apiKey: "",
    authDomain: "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: ""
};
````
